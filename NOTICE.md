# Notice

Cipher Snagem Editor is a C#/.NET/Avalonia remake of the legacy Pokemon
Colosseum and Pokemon XD: Gale of Darkness tools originally created by **Stars
Momodu** / **@StarsMMD**.

The original project is commonly known through:

- `GoD Tool`
- `Colosseum Tool`
- `Pokemon-XD-Code`
- `https://github.com/PekanMmd/Pokemon-XD-Code.git`

This repository was built by studying the original Swift source, data models,
binary parsers, UI layouts, assets, and practical workflows. It is intended as a
preservation-minded Windows/Linux fork/remake, not as an attempt to obscure the
origin of the work.

The repository uses GPL-2.0-only licensing metadata to remain compatible with
the legacy source. See `LICENSE`.

Pokemon, Pokemon Colosseum, Pokemon XD: Gale of Darkness, Genius Sonority,
Nintendo, and related names are owned by their respective rights holders. This
project is unofficial fan tooling and does not include game files.
